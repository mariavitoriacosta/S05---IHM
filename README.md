# 📱 Redesign do App Acadêmico do Inatel

## Descrição

Este projeto foi desenvolvido na disciplina de Interação Homem-Máquina (IHM) com o objetivo de melhorar a experiência do usuário no aplicativo acadêmico do Inatel.

A proposta consistiu em criar uma interface mais intuitiva, organizada e responsiva, além de adicionar uma nova funcionalidade de calendário personalizado para auxiliar os alunos na organização de suas atividades acadêmicas.

---

## Objetivo

O principal objetivo do projeto foi aplicar conceitos de IHM para melhorar a usabilidade do sistema, tornando a navegação mais simples e eficiente, além de oferecer uma ferramenta útil para o dia a dia dos estudantes.

---

## Funcionalidades

O sistema possui as seguintes funcionalidades:

* Interface mais intuitiva e moderna
* Layout responsivo
* Modo escuro (dark mode)
* Calendário personalizado
* Adição de eventos no calendário:

  * Provas
  * Trabalhos
  * Lembretes
* Organização visual das informações acadêmicas

## Destaques do Projeto

### Modo Escuro

O sistema conta com suporte a dark mode, melhorando a experiência do usuário em ambientes com pouca luz.

---

### Calendário Personalizado

Foi implementado um calendário interativo que permite ao usuário adicionar eventos importantes, como provas e trabalhos, facilitando a organização acadêmica.

---

### Foco em IHM

Durante o desenvolvimento, foram aplicados conceitos importantes da área de Interação Homem-Máquina, como:

* Usabilidade
* Clareza na interface
* Organização visual
* Feedback ao usuário
* Acessibilidade básica

## Observações

* O projeto é totalmente front-end
* Não é necessário instalar dependências
* Os dados utilizados (como aulas) estão no arquivo `aulas.json`

## Considerações Finais

Este projeto foi importante para compreender, na prática, como pequenas mudanças no design e na organização da interface podem impactar diretamente a experiência do usuário.

Além disso, a implementação do calendário trouxe uma funcionalidade útil que agrega valor ao sistema acadêmico.
